功能：对多项式化简求值求导。

使用说明： 
输入->x*x + 2y 
输入->!simplify x=2 
显示->4+2y 
输入->!d/d x 
显示->2*x+2*y

可输入的表达式限制： 可以有空格， 在出现"+"的位置可以使用"-", 多项式的变量可以使用长度大于1的字符串表示， 可以使用^表示幂运算，但仅出现在变量后面，且仅为整数， 无小数和括号。

可输入的表达式形如: 5+2或0x-y^2或5xoo^2+25-x^2y或3 5 + 2 x2 - 3y^2*2
